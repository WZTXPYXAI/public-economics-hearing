# X市建投集团300亿债务危机紧急听证会

这是一个面向本科公共经济学课程的互动角色扮演网页项目。项目基于“X市建投集团300亿债务危机紧急听证会”场景，模拟地方政府债务风险、中央财政兜底、隐性担保、刚性兑付、纳税人负担与金融稳定之间的政策冲突。

项目采用纯前端实现，可直接部署在 GitHub Pages 上运行。

## 项目背景

省会 X 市建投集团总债务约 1500 亿元，其中 300 亿元将在半年内集中到期。该集团承担水务、供暖、地铁、棚改等公共民生业务。

如果债务违约，可能导致 300 万市民公共服务受到影响，并引发全省城投债务连锁风险。省政府因此向中央财政部申请发行 300 亿特别再融资债，用于化解本次到期债务。

本项目通过角色扮演方式，引导学生思考：

- 中央财政是否应该兜底地方城投债务？
- 金融稳定和财政纪律如何权衡？
- 隐性担保是否应被打破？
- 全国纳税人是否应承担地方债务成本？
- 公共服务连续性与预算约束之间如何平衡？

## 角色设置

网页提供 5 个可选择角色：

1. **省政府｜求救派**  
   主张中央批准特别再融资债，避免城投债务风险外溢。

2. **财政部｜硬汉派**  
   强调《预算法》纪律，反对形成中央兜底预期。

3. **城投集团负责人｜委屈派**  
   强调城投承担公共任务，自身并非完全自主举债。

4. **债券债权人｜刚兑派**  
   主张维护债券兑付，避免信用债市场恐慌。

5. **普通纳税人｜问责派**  
   质疑权责不对等，反对全国纳税人为地方债务买单。
